# linuxcontainers.org

This branch contains the content and site generator of the
linuxcontainers.org website.

## Dependencies

 * man2html-base
 * python3 (>= 3.4)
 * python3-bs4
 * python3-jinja2
 * python3-markdown
 * python3-pygments

## Generating the website

    ./generate
